# count-svn-commits

Simple python script to count how many commits each author has inside a svn repo

## Usage

`python3 count.py SVN_DIR`

## Sample Output

```shell
$ python3 count.py projects/count-svn-commits
SVN_DIR: projects/count-svn-commits

author1   2
author2   10
author3   22
TOTAL     34
```
